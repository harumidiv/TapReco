# アプリについて
たぷれこは録音アプリです、録音開始ボタンが画面全域なので緊急時にもポケットの中などで録音を開始することができます。
また、録音停止もスライド式のボタンを採用しているので間違えて録音停止してしまう事故を予防できます。
